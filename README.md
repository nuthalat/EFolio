# EFolio

Personal portfolio website for **Tejasvi Nuthalapati**.

## Preview the website (not raw files)

### Option 1: Quick local preview (recommended)
From the project folder, run:

```bash
python -m http.server 8000
```

Then open this URL in your browser:

- <http://localhost:8000>

This renders the site exactly like a webpage.

### Option 2: Open directly
You can also open `index.html` directly in a browser, but serving it with Python is better for a realistic preview.

## Files to edit

- `index.html` → page structure and text content
- `styles.css` → colors, spacing, and visual design
- `script.js` → project cards and dynamic content
